# 台灣觀光查詢網
nodejs-version：v12.14.0

本網站整合台灣交通部 TDX API，提供即時的旅宿景點資訊、YouBike 借還點與停車位狀態、公車即時動態與路線圖，幫助使用者掌握最新交通與觀光資訊。

本專案使用 Vue 3、SCSS、Vuex 開發，並透過 Leaflet + Wicket 將公車路線精準繪製在地圖上，提供直覺的視覺化交通資訊，提升使用體驗。

技術：Vue3、Vuex、SCSS、RWD、Leaflet.js、TDX api
